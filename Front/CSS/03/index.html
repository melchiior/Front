CSS Colors

Cores no CSS devem seguir os seguintes padrões:

  • Hexadecimal
  • Hexadecimal colors com transparencia
  • RGB
  • RGBA
  • HSL
  • HSLA
  • Cores predefinidas pelo navegador

RGB

Cores no formato RGB detem dentre parenteses 3 numeros:

O primeiro para VERMELHO (RED)
O segundo para VERDE (GREEN)
O terceiro para AZUL (BLUE)

Formando todas as demais cores com a combinação desas 3.

Hexadecimal

Cores no formato Hexadecial utilizando a inicial # (jogo da velha) onde os 2 primeiros digitos são para o Vermelho, após 2 dígitos para Verde, e 2 últimos dígitos para Azul.

Extremamente semelhante ao RGB

HSL

Cores no formato HSL utilizam a combinação de 3 valores:
  • Hue
    valor variavel entre 0 e 360, onde ate 120 vermelho, entre 120 e 240 verde e 240 para azul
  • Saturação
    variavel entre 0% e 100%
  • Brilho
    variavel entre 0% e 100%
